## Vercel Deployment

### Option 1: Deploy via Web (Recommended)

1. Go to: https://vercel.com/new
2. Import Git Repository ‚Üí Select `TamarCoder/WebForm`
3. Configure:
   - Framework Preset: **Next.js**
   - Root Directory: **.**
   - Build Command: `npm run build`
   - Output Directory: `.next`
4. Click **Deploy**
5. Copy the URL and update README.md

### Option 2: Deploy via CLI

```bash
# Install Vercel CLI
npm install -g vercel

# Login
vercel login

# Deploy
cd ~/Desktop/WebForm
vercel

# Follow prompts:
# - Set up and deploy? Yes
# - Which scope? Your account
# - Link to existing project? No
# - What's your project's name? webform
# - In which directory is your code located? ./
# - Auto-detected settings? Yes
# - Deploy? Yes

# For production:
vercel --prod
```
