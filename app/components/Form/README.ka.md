# ­ЪЊЮ Form рЃЎрЃЮрЃЏрЃърЃЮрЃюрЃћрЃюрЃбрЃў

**­ЪЄг­ЪЄф рЃЦрЃљрЃарЃЌрЃБрЃџрЃў** | [­ЪЄг­ЪЄД English](./README.md)

рЃЏрЃЌрЃљрЃЋрЃљрЃарЃў рЃЏрЃарЃљрЃЋрЃљрЃџрЃћрЃбрЃљрЃърЃўрЃљрЃюрЃў рЃарЃћрЃњрЃўрЃАрЃбрЃарЃљрЃфрЃўрЃўрЃА рЃцрЃЮрЃарЃЏрЃўрЃА рЃЎрЃЮрЃЏрЃърЃЮрЃюрЃћрЃюрЃбрЃў, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃљрЃгрЃДрЃЮрЃЉрЃА рЃЏрЃЌрЃћрЃџ рЃарЃћрЃњрЃўрЃАрЃбрЃарЃљрЃфрЃўрЃўрЃА рЃърЃарЃЮрЃфрЃћрЃАрЃА.

---

## ­ЪЊќ рЃЏрЃўрЃЏрЃЮрЃ«рЃўрЃџрЃЋрЃљ

Form рЃЎрЃЮрЃЏрЃърЃЮрЃюрЃћрЃюрЃбрЃў рЃљрЃарЃўрЃА рЃфрЃћрЃюрЃбрЃарЃљрЃџрЃБрЃарЃў рЃЮрЃарЃЦрЃћрЃАрЃбрЃарЃљрЃбрЃЮрЃарЃў рЃЏрЃарЃљрЃЋрЃљрЃџрЃћрЃбрЃљрЃърЃўрЃљрЃюрЃў рЃарЃћрЃњрЃўрЃАрЃбрЃарЃљрЃфрЃўрЃўрЃА рЃърЃарЃЮрЃфрЃћрЃАрЃўрЃАрЃЌрЃЋрЃўрЃА. рЃЏрЃљрЃарЃЌрЃљрЃЋрЃА state-рЃА рЃАрЃљрЃЏ рЃћрЃбрЃљрЃърЃќрЃћ, рЃљрЃЏрЃБрЃерЃљрЃЋрЃћрЃЉрЃА рЃюрЃљрЃЋрЃўрЃњрЃљрЃфрЃўрЃљрЃА рЃЊрЃљ рЃљрЃћрЃарЃЌрЃўрЃљрЃюрЃћрЃЉрЃА рЃДрЃЋрЃћрЃџрЃљ рЃЦрЃЋрЃћ-рЃЎрЃЮрЃЏрЃърЃЮрЃюрЃћрЃюрЃбрЃА (EmailSection, SummarySection, SignIn) рЃћрЃарЃЌрЃўрЃљрЃю рЃЏрЃЮрЃЏрЃ«рЃЏрЃљрЃарЃћрЃЉрЃџрЃўрЃА рЃњрЃљрЃЏрЃЮрЃфрЃЊрЃўрЃџрЃћрЃЉрЃљрЃерЃў.

---

## Рюе рЃцрЃБрЃюрЃЦрЃфрЃўрЃћрЃЉрЃў

- РюЁ 3-рЃћрЃбрЃљрЃърЃўрЃљрЃюрЃў рЃарЃћрЃњрЃўрЃАрЃбрЃарЃљрЃфрЃўрЃўрЃА рЃърЃарЃЮрЃфрЃћрЃАрЃў
- РюЁ рЃЊрЃўрЃюрЃљрЃЏрЃўрЃБрЃарЃў рЃюрЃљрЃЉрЃўрЃ»рЃўрЃА рЃўрЃюрЃЊрЃўрЃЎрЃљрЃбрЃЮрЃарЃћрЃЉрЃў рЃърЃарЃЮрЃњрЃарЃћрЃАрЃўрЃА рЃЌрЃЋрЃљрЃџрЃЌрЃЋрЃљрЃџрЃўрЃЌ
- РюЁ рЃцрЃЮрЃарЃЏрЃўрЃА state рЃЏрЃљрЃарЃЌрЃЋрЃљ рЃћрЃбрЃљрЃърЃћрЃЉрЃќрЃћ
- РюЁ React Hook Form рЃўрЃюрЃбрЃћрЃњрЃарЃљрЃфрЃўрЃљ
- РюЁ Yup schema рЃЋрЃљрЃџрЃўрЃЊрЃљрЃфрЃўрЃљ
- РюЁ рЃњрЃљрЃЊрЃљрЃарЃЌрЃЋрЃљ рЃерЃћрЃАрЃЋрЃџрЃљрЃАрЃљ рЃЊрЃљ рЃарЃћрЃњрЃўрЃАрЃбрЃарЃљрЃфрЃўрЃљрЃА рЃерЃЮрЃарЃўрЃА
- РюЁ рЃАрЃарЃБрЃџрЃДрЃЮрЃцрЃўрЃџрЃў рЃцрЃЮрЃарЃЏрЃўрЃА рЃЋрЃћрЃџрЃћрЃЉрЃў
- РюЁ TypeScript рЃЏрЃ«рЃљрЃарЃЊрЃљрЃГрЃћрЃарЃљ
- РюЁ SCSS рЃЏрЃЮрЃЊрЃБрЃџрЃўрЃА рЃАрЃбрЃўрЃџрЃўрЃќрЃљрЃфрЃўрЃљ

---

## ­Ъј» рЃарЃћрЃњрЃўрЃАрЃбрЃарЃљрЃфрЃўрЃўрЃА рЃърЃарЃЮрЃфрЃћрЃАрЃў

### рЃюрЃљрЃЉрЃўрЃ»рЃў 1: рЃФрЃўрЃарЃўрЃЌрЃљрЃЊрЃў рЃўрЃюрЃцрЃЮрЃарЃЏрЃљрЃфрЃўрЃљ
рЃљрЃњрЃарЃЮрЃЋрЃћрЃЉрЃА рЃЏрЃЮрЃЏрЃ«рЃЏрЃљрЃарЃћрЃЉрЃџрЃўрЃА рЃърЃўрЃарЃљрЃЊ рЃЊрЃљ рЃърЃарЃЮрЃцрЃћрЃАрЃўрЃБрЃџ рЃЊрЃћрЃбрЃљрЃџрЃћрЃЉрЃА:
- рЃАрЃарЃБрЃџрЃў рЃАрЃљрЃ«рЃћрЃџрЃў
- рЃАрЃљрЃљрЃарЃЕрЃћрЃЋрЃюрЃЮ рЃЮрЃџрЃЦрЃў
- рЃърЃарЃЮрЃњрЃарЃљрЃЏрЃўрЃарЃћрЃЉрЃўрЃА рЃЏрЃўрЃЏрЃљрЃарЃЌрЃБрЃџрЃћрЃЉрЃљ (Dropdown)
- рЃњрЃљрЃЏрЃЮрЃфрЃЊрЃўрЃџрЃћрЃЉрЃўрЃА рЃЊрЃЮрЃюрЃћ (Dropdown)
- рЃАрЃљрЃЏрЃБрЃерЃљрЃЮрЃА рЃЊрЃљрЃгрЃДрЃћрЃЉрЃўрЃА рЃЌрЃљрЃарЃўрЃдрЃў (рЃЎрЃљрЃџрЃћрЃюрЃЊрЃљрЃарЃў)
- рЃАрЃЦрЃћрЃАрЃў (рЃарЃљрЃЊрЃўрЃЮ рЃдрЃўрЃџрЃљрЃЎрЃћрЃЉрЃў)
- рЃерЃћрЃбрЃДрЃЮрЃЉрЃўрЃюрЃћрЃЉрЃљ (Textarea)
- **рЃњрЃљрЃюрЃљрЃЌрЃџрЃћрЃЉрЃўрЃА рЃАрЃћрЃЦрЃфрЃўрЃљ:**
  - рЃ«рЃљрЃарЃўрЃАрЃ«рЃў
  - рЃўрЃюрЃАрЃбрЃўрЃбрЃБрЃбрЃў
  - рЃЊрЃљрЃЏрЃЌрЃљрЃЋрЃарЃћрЃЉрЃўрЃА рЃЌрЃљрЃарЃўрЃдрЃў

### рЃюрЃљрЃЉрЃўрЃ»рЃў 2: рЃљрЃюрЃњрЃљрЃарЃўрЃерЃўрЃА рЃерЃћрЃЦрЃЏрЃюрЃљ
рЃљрЃЏрЃБрЃерЃљрЃЋрЃћрЃЉрЃА EmailSection рЃЎрЃЮрЃЏрЃърЃЮрЃюрЃћрЃюрЃбрЃў:
- рЃћрЃџ-рЃцрЃЮрЃАрЃбрЃљ
- рЃърЃљрЃарЃЮрЃџрЃў
- рЃћрЃџ-рЃцрЃЮрЃАрЃбрЃўрЃА рЃЊрЃљрЃЊрЃљрЃАрЃбрЃБрЃарЃћрЃЉрЃљ

### рЃюрЃљрЃЉрЃўрЃ»рЃў 3: рЃерЃћрЃ»рЃљрЃЏрЃћрЃЉрЃљ рЃЊрЃљ рЃњрЃљрЃЊрЃљрЃ«рЃћрЃЊрЃЋрЃљ
рЃљрЃЏрЃБрЃерЃљрЃЋрЃћрЃЉрЃА SummarySection рЃЎрЃЮрЃЏрЃърЃЮрЃюрЃћрЃюрЃбрЃў:
- рЃДрЃЋрЃћрЃџрЃљ рЃерЃћрЃДрЃЋрЃљрЃюрЃўрЃџрЃў рЃўрЃюрЃцрЃЮрЃарЃЏрЃљрЃфрЃўрЃўрЃА рЃњрЃљрЃЊрЃљрЃ«рЃћрЃЊрЃЋрЃљ
- рЃАрЃљрЃЉрЃЮрЃџрЃЮрЃЮ рЃњрЃљрЃњрЃќрЃљрЃЋрЃюрЃљ

## ­ЪЊд рЃЏрЃЌрЃљрЃЋрЃљрЃарЃў рЃцрЃЮрЃарЃЏрЃўрЃА рЃЋрЃћрЃџрЃћрЃЉрЃў

### рЃърЃўрЃарЃљрЃЊрЃў рЃўрЃюрЃцрЃЮрЃарЃЏрЃљрЃфрЃўрЃљ
- `fullName`: string (рЃљрЃБрЃфрЃўрЃџрЃћрЃЉрЃћрЃџрЃў)
- `constituency`: string (рЃљрЃБрЃфрЃўрЃџрЃћрЃЉрЃћрЃџрЃў)
- `programmingField`: dropdown (рЃљрЃБрЃфрЃўрЃџрЃћрЃЉрЃћрЃџрЃў)
- `position`: dropdown (рЃљрЃБрЃфрЃўрЃџрЃћрЃЉрЃћрЃџрЃў)
- `dateOfBirth`: Date (рЃљрЃБрЃфрЃўрЃџрЃћрЃЉрЃћрЃџрЃў, рЃБрЃюрЃЊрЃљ рЃўрЃДрЃЮрЃА рЃгрЃљрЃарЃАрЃБрЃџрЃў рЃЌрЃљрЃарЃўрЃдрЃў)
- `gender`: radio (рЃљрЃБрЃфрЃўрЃџрЃћрЃЉрЃћрЃџрЃў)
- `message`: textarea (рЃљрЃарЃЕрЃћрЃЋрЃўрЃЌрЃў)

### рЃњрЃљрЃюрЃљрЃЌрЃџрЃћрЃЉрЃљ
- `degree`: string (рЃљрЃБрЃфрЃўрЃџрЃћрЃЉрЃћрЃџрЃў)
- `institution`: string (рЃљрЃБрЃфрЃўрЃџрЃћрЃЉрЃћрЃџрЃў)
- `graduationDate`: Date (рЃљрЃБрЃфрЃўрЃџрЃћрЃЉрЃћрЃџрЃў)

  programmingField: yup.string().required("рЃњрЃЌрЃ«рЃЮрЃЋрЃЌ рЃљрЃўрЃарЃЕрЃўрЃЮрЃЌ рЃърЃарЃЮрЃњрЃарЃљрЃЏрЃўрЃарЃћрЃЉрЃўрЃА рЃЏрЃўрЃЏрЃљрЃарЃЌрЃБрЃџрЃћрЃЉрЃљ"),

  message: yup.string(),
  degree: yup.string().required("рЃ«рЃљрЃарЃўрЃАрЃ«рЃў рЃљрЃБрЃфрЃўрЃџрЃћрЃЉрЃћрЃџрЃўрЃљ"),

## ­Ъје рЃюрЃљрЃЉрЃўрЃ»рЃўрЃА рЃўрЃюрЃЊрЃўрЃЎрЃљрЃбрЃЮрЃарЃћрЃЉрЃў

рЃцрЃЮрЃарЃЏрЃљ рЃљрЃЕрЃЋрЃћрЃюрЃћрЃЉрЃА рЃЋрЃўрЃќрЃБрЃљрЃџрЃБрЃа рЃюрЃљрЃЉрЃўрЃ»рЃўрЃА рЃўрЃюрЃЊрЃўрЃЎрЃљрЃбрЃЮрЃарЃћрЃЉрЃА рЃќрЃћрЃЏрЃЮрЃЌ:
- **рЃюрЃљрЃЉрЃўрЃ»рЃў 1:** рЃФрЃўрЃарЃўрЃЌрЃљрЃЊрЃў рЃўрЃюрЃцрЃЮрЃарЃЏрЃљрЃфрЃўрЃљ (рЃюрЃљрЃарЃўрЃюрЃ»рЃўрЃАрЃцрЃћрЃарЃў рЃарЃЮрЃфрЃљ рЃљрЃЦрЃбрЃўрЃБрЃарЃў/рЃЊрЃљрЃАрЃарЃБрЃџрЃћрЃЉрЃБрЃџрЃў)
- **рЃюрЃљрЃЉрЃўрЃ»рЃў 2:** рЃћрЃџ-рЃцрЃЮрЃАрЃбрЃљ рЃЊрЃљ рЃърЃљрЃарЃЮрЃџрЃў (рЃюрЃљрЃарЃўрЃюрЃ»рЃўрЃАрЃцрЃћрЃарЃў рЃарЃЮрЃфрЃљ рЃљрЃЦрЃбрЃўрЃБрЃарЃў/рЃЊрЃљрЃАрЃарЃБрЃџрЃћрЃЉрЃБрЃџрЃў)
- **рЃюрЃљрЃЉрЃўрЃ»рЃў 3:** рЃерЃћрЃ»рЃљрЃЏрЃћрЃЉрЃљ (рЃюрЃљрЃарЃўрЃюрЃ»рЃўрЃАрЃцрЃћрЃарЃў рЃарЃЮрЃфрЃљ рЃљрЃЦрЃбрЃўрЃБрЃарЃў)
