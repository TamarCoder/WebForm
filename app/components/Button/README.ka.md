# 🎯 ღილაკის კომპონენტი

**🇬🇪 ქართული** | [🇬🇧 English](./README.md)

მორგებადი, მრავალჯერადი გამოყენების ღილაკის კომპონენტი TypeScript მხარდაჭერითა და SCSS მოდულებით.

---

## 📖 მიმოხილვა

Button კომპონენტი არის მოქნილი, ხელმისაწვდომი ღილაკი, რომელიც მხარს უჭერს მორგებულ ტექსტს, დაწკაპუნების დამუშავებას და სტილიზაციას. შექმნილია აპლიკაციის მასშტაბით გამოსაყენებლად თანმიმდევრული UI/UX-ისთვის.

---

## ✨ ფუნქციები

- ✅ TypeScript მხარდაჭერა
- ✅ მორგებული ტექსტი და დაწკაპუნების დამუშავება
- ✅ SCSS მოდულის სტილიზაცია
- ✅ ხელმისაწვდომი და სემანტიკური HTML
- ✅ მრავალჯერადი გამოყენება აპლიკაციაში

---

## 📦 Props

| Prop | ტიპი | აუცილებელი | ნაგულისხმევი | აღწერა |
|------|------|------------|---------------|--------|
| `text` | `string` | კი | - | ღილაკის ტექსტი |
| `onClick` | `() => void` | არა | - | დაწკაპუნების მოვლენის დამუშავება |
| `type` | `"button" \| "submit" \| "reset"` | არა | `"button"` | ღილაკის HTML ტიპი |

---

## 🚀 გამოყენება

### ძირითადი მაგალითი

```tsx
import Button from "@/app/components/Button/Button";

function MyComponent() {
  return (
    <Button text="დააწკაპუნე" onClick={() => console.log("დაიწკაპუნა!")} />
  );
}
```

### Submit ღილაკი

```tsx
<Button text="გაგზავნა" type="submit" />
```

### დაწკაპუნების დამუშავების გარეშე

```tsx
<Button text="გაუქმება" />
```

## 🎨 სტილიზაცია

კომპონენტი იყენებს SCSS მოდულებს სკოპირებული სტილიზაციისთვის. მოირგეთ გარეგნობა შეცვლით:

```scss
// Button.module.scss
.button {
  // თქვენი მორგებული სტილები აქ
}
```

## ♿ ხელმისაწვდომობა

- სემანტიკური `<button>` ელემენტი
